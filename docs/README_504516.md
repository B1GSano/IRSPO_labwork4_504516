# Geometry Functions

## Общее описание  

Проект состоит из двух модулей:  

- **`circle.py`** — функции для вычисления площади и длины окружности по радиусу.  
- **`square.py`** — функции для вычисления площади и периметра квадрата по стороне.  

Каждая функция документирована и сопровождается примерами использования.  

---

## Модуль `circle.py`

### `area(r)`  
Вычисляет площадь круга по радиусу.  

**Формула:** 
\[
S = pi * r^2
\]


**Пример:**  
```python
from circle import area

print(area(1))   # 3.141592653589793
print(area(5))   # 78.53981633974483
```
---

## Модуль `square.py`

### `area(a)`
Вычисляет площадь квадрата по стороне

**Формула:** 
\[
S = a^2
\]

### `perimeter(a)`
Вычисляет периметер квадрата по стороне

**Формула:** 
\[
S = a * 4
\]

**Пример:**  
```python
from square import area
from square import perimeter

print(area(1))   # 1
print(area(5))   # 25
print(perimeter(1))   # 4
print(perimeter(5))   # 20
```
