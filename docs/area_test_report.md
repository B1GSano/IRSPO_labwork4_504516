
# Отчет о тестировании функции `area`

## 1. Цели и задачи тестирования
Цель: проверить корректность функции вычисления площади квадрата.  
Задачи:
- проверить результат при `a = 0`;
- проверить результат при `a = 10`;
- убедиться, что тесты выполняются без ошибок.

## 2. Описание тестируемого продукта
Тестируется функция:
```python
def area(a):
    return a * a
```
Назначение: вернуть площадь квадрата со стороной `a`.


## 3. Область тестирования
Покрытые входы:
- граничный случай: `a = 0` → ожидается `0`;
- типичный случай: `a = 10` → ожидается `100`.

Не покрыто: отрицательные числа, типы кроме `int`/`float`.

## 4. Стратегия тестирования
Метод: модульное тестирование.  
Инструмент: стандартный модуль `unittest`.  
Подход: сравнение ожидаемого и фактического значений.

Код теста:
```python
import unittest

from geometric_lib.square import area


class SquareTestCase(unittest.TestCase):
	def test_zero_mul(self):
		res = area(0)
		self.assertEqual(res, 0)

	def test_square_mul(self):
		res = area(10)
		self.assertEqual(res, 100)
```

## 5. Критерии приемки
- `area(0) == 0`;
- `area(10) == 100`;
- запуск `unittest` возвращает статус `OK`.

## 6. Ожидаемые результаты
При запуске:
```bash
python -m unittest имя_файла.py
```
ожидается вывод:
```text
..
----------------------------------------------------------------------
Ran 2 tests in 0.000s

OK
```

## 7. Вывод
Функция `area` проходит модульные тесты для заданных входных данных.
